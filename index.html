QxStack Auto-Deployment Tool   2016-present   Patent

In order to facilitate the deployment of QCT solutions, the QxStack Auto-Deployment Tool provides a streamlined process for hardware introspection and provisioning, solution deployment, and solution validation. With an easy-to-use web interface, this tool can dramatically reduce time and minimize guest effort required for building private cloud solutions. In this project, I not only responsible for the development of Ansible-based auto-deployment process, but also play a role in technical project management to evaluate the project schedule, resources and deliverables.
